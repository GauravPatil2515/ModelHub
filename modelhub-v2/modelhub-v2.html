How it works
From local model to live API in 4 steps
01
Train locally
Build any scikit-learn model in Colab or Kaggle. Save it with joblib.dump().
02
Upload here
Drag and drop your .pkl file. Fill in a short model card — name, description, tags, author.
03
Auto-deploy
Backend validates the model, runs a test prediction, and auto-uploads to your S3 bucket via boto3.
04
Share & run
A public model card is created instantly. Anyone can run inference from the browser or via the REST API.
